# MIBs

Coloque aqui a MIB do gateway.

Depois configure no `.env`:

```env
SNMP_MIB_DIR=app/mibs
SNMP_MIB_NAME=SYNWAY-GW-MIB
SNMP_LINE_1_OID=...
SNMP_LINE_2_OID=...
SNMP_LINE_3_OID=...
SNMP_LINE_4_OID=...
SNMP_LINE_5_OID=...
SNMP_LINE_6_OID=...
SNMP_LINE_7_OID=...
SNMP_LINE_8_OID=...
```

Nesta instalacao, a MIB `SYNWAY-GW-MIB.txt` foi adicionada a esta pasta.
Os OIDs podem ser numericos ou nomes resolvidos pela MIB quando suportados pelo
PySNMP.
